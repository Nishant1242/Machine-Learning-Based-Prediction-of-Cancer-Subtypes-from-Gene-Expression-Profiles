# Machine Learning–Based Prediction of Cancer Subtypes from Gene Expression Profiles

This repository contains code and analysis for a bioinformatics project that uses  
microarray gene expression data to classify leukemia subtypes (ALL vs AML) using machine learning.

The primary goals are:
- To build reproducible pipelines for preprocessing and analyzing high-dimensional gene expression data.
- To train and evaluate machine learning models for subtype classification.
- To identify genes that are most predictive of leukemia subtype.

---

## 1. Project Overview

Acute Lymphoblastic Leukemia (ALL) and Acute Myeloid Leukemia (AML) have distinct molecular profiles and clinical outcomes.  
Gene expression profiling provides a high-dimensional view of transcriptional activity that can be leveraged to distinguish these subtypes.

This project uses the classic Golub leukemia microarray dataset (Kaggle version) to:
- Preprocess and normalize gene expression data.
- Visualize sample structure using PCA.
- Train baseline and advanced machine learning models.
- Evaluate performance and explore gene-level importance.

---

## 2. Data

**Source:**  
Kaggle – Golub Leukemia Gene Expression dataset (microarray-based).

**Files used (stored in `data/raw/`):**
- `data_set_ALL_AML_train.csv`
- `data_set_ALL_AML_independent.csv`
- `actual.csv` (patient labels: ALL vs AML)

**Processed file (generated by the notebook):**
- `data/processed/all_aml_expression_processed.csv`  
  - Rows: 72 samples  
  - Columns: 7129 genes + `cancer` label

> Note: Raw data is not generated by this project; it is downloaded externally and placed into `data/raw/`.
